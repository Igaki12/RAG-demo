ニュース理解アプリの目次

ガラスモーフィズムをまとった各種 Web アプリの入り口です。ユーザーが時事ニュースを読み解いたり、ベクトル検索で記事を探したりする体験をサッと比較できます。

共通点

  • ニュース記事の JSONL データを読み込み、共起ネットワークで関係性を可視化。
  • クイズ出題と記事本文表示を行き来しながら、理解を深められる学習体験を提供。
  • バックグラウンドでカード UI とガラスモーフィズムを採用し、視覚的に統一。

違いの見どころ

  • Commonsense ver8 は日付タイムラインとランダム出題を備えた探索重視の体験。
  • Current Affairs Quest v1 はクイズ先行で根拠記事への導線を切り替えられる二段構成。
  • 学習フローの入り口が異なり、復習や探索のリズムを好みに合わせて選べます。

体験できるアプリ

Commonsense ver8

Tailwind を取り入れた最新 UI。アップロードしたニュースを日付ごとに可視化し、記事モーダル内で理解度テストを進められます。

  • 共通共起ネットワークとクイズの往復でニュース理解を深める設計。
  • 違いランダム出題モーダルやアニメーションを加え、探索型の学習サイクルを強化。

Current Affairs Quest

日付範囲を指定してランダムクイズを解き、必要に応じてネットワーク図へ切り替える二段構成アプリ。クイズから根拠記事に素早く遷移できます。

  • 共通JSONL データから記事本文を抽出し、モーダルで全容を確認できます。
  • 違いクイズを先に提示し、回答前に「根拠を探す」か「答えを見る」かを選べるフローを備えています。

2 つのニュース学習アプリをこのページからすぐに試せます。
